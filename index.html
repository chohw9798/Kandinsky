홈페이지의 성격

  • 키보드와 마우스를 통해 사용자와의 상호작용이 가능하다.
  • 유명한 화가의 그림을 '조희원'이 모방하여 새롭게 만든 작품을 전시해 놓은 홈페이지이다.



작품 제목 : 여러 개의 원

원작 화가 : 바실리 칸딘스키

  • 이 작품은 칸딘스키의 작품을 모방한 작품으로, 각각의 원의 투명도를 다르게 하여 원끼리 겹쳐진 부분의 색이 잘 표현되도록 하였다.
  • Interaction 1 : 키보드의 상, 하, 좌, 우 키를 누르면 각각의 원의 상, 하, 좌, 우에 똑같은 원이 하나 더 그려진다.
  • Interaction 2 : 마우스 왼쪽 버튼을 누르면 랜덤한 투명도와 빨간색 계열의 색으로 지름 100이하의 랜덤한 크기의 원이 마우스 위치에 그려진다.
  • Interaction 3 : 마우스 가운데 버튼을 누르면 랜덤한 투명도와 파란색 계열의 색으로 지름 100이하의 랜덤한 크기의 원이 마우스 위치에 그려진다.
